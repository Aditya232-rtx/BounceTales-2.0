## ğŸ® How to Play

1. **Starting the Game**
   ```bash
   python main.py
   ```

2. **Main Menu**
   - **Start Game**: Begin your adventure
   - **Select Level**: Choose a specific level to play
   - **Customize**: Personalize your ball's appearance
   - **Quit**: Exit the game

3. **Game Controls**
   - **Left/Right Arrow Keys**: Move the ball
   - **Space**: Jump
   - **R**: Reset current level
   - **ESC**: Pause and show menu
